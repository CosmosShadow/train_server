# train_server

### requirements

[lake](https://github.com/CosmosShadow/lake)


### 使用

#### 配制

```
mv data/projects_template.csv data/projects.csv
```

然后到projects.csv中添加需要查看的项目

#### 启动

```powershell
python server.py

# the open http://0.0.0.0:8080/ on your browser
```
